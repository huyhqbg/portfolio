Click [here](https://storage-exercise.s3.ap-southeast-1.amazonaws.com/out/index.html) to see the demo.

### Run this project
1. Clone the project
```bash
git clone https://github.com/trongcho34/portfolio.git
cd portfolio
```

2. Start the project
- using `npm`
```bash
npm install
npm run dev
```
- or using `yarn`
```bash
yarn
yarn dev
```
