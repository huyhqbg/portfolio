## portfolio

### About
This is a cloned Portfolio UI. [Here](https://portfolio.framer.website/) is the link of the website.

### Run this project
1. Clone the project
```bash
git clone https://github.com/trongcho34/portfolio.git
cd portfolio
```

2. Start the project
- using `npm`
```bash
npm install
npm run dev
```
- or using `yarn`
```bash
yarn
yarn dev
```
